Yoga with Jib — Class Tracker App

This application is a convenience tool for tracking prepaid yoga class packages. By using this app, you agree to the following:

1. Data & Privacy
This app stores only usernames and class counts. No full names, ID numbers, or phone numbers should be entered.

2. Payments
Always verify bank details directly with your teacher before transferring. Yoga with Jib accepts no liability for incorrect payments.

3. Class Records
Records are maintained in good faith. Discrepancies should be resolved between teacher and student.

4. Service Availability
This app is provided "as is" without warranty.

Last updated: April 2026

Yoga with Jib — แอปติดตามคลาสโยคะ

แอปพลิเคชันนี้ ("Tally by Yoga with Jib") จัดทำขึ้นเพื่อเป็นเครื่องมือช่วยติดตามแพ็คเกจคลาสโยคะที่ชำระเงินล่วงหน้า การใช้แอปนี้ถือว่าคุณรับทราบและยอมรับข้อกำหนดต่อไปนี้

1. ข้อมูลและความเป็นส่วนตัว
แอปนี้จัดเก็บเฉพาะชื่อเล่น (username) และจำนวนคลาสเท่านั้น ห้ามใส่ชื่อจริง เลขบัตรประชาชน หมายเลขโทรศัพท์ หรือข้อมูลส่วนตัวที่ละเอียดอ่อน ผู้พัฒนาไม่รับผิดชอบต่อข้อมูลที่ผู้ใช้กรอกเข้ามาโดยขัดต่อแนวทางนี้

2. การชำระเงิน
กรุณายืนยันข้อมูลบัญชีกับครูโดยตรงก่อนโอนเงินทุกครั้ง Yoga with Jib ไม่รับผิดชอบต่อการโอนเงินผิดบัญชีหรือข้อพิพาททางการเงินใดๆ ที่เกิดจากการใช้แอปนี้

3. บันทึกคลาส
การบันทึกดำเนินการด้วยความสุจริต หากมีความคลาดเคลื่อนใดๆ ให้ครูและนักเรียนตกลงกันโดยตรง Yoga with Jib ไม่รับผิดชอบต่อการสูญหายของข้อมูลหรือความไม่ถูกต้อง

4. ความพร้อมใช้งาน
แอปนี้ให้บริการ "ตามสภาพที่เป็น" โดยไม่มีการรับประกัน Yoga with Jib ไม่รับประกันความเสถียรหรือความต่อเนื่องของบริการ

5. บริการของบุคคลที่สาม
แอปนี้เชื่อมต่อกับ Google Apps Script, Firebase และ GitHub Pages ซึ่งอยู่ภายใต้ข้อกำหนดของแต่ละผู้ให้บริการ Yoga with Jib ไม่รับผิดชอบต่อความพร้อมใช้งานหรือการจัดการข้อมูลของบริการเหล่านั้น

🚀 First-time setup
Complete these steps once to get the app ready
The default password is 1242 — change it immediately.
Tap ← Back to return to the login screen → tap Change admin password.
Use at least 8 characters with a mix of letters and numbers.
From the home screen → tap I'm a teacher → tap + I'm a new teacher.
Fill in display name, choose a username (cannot be changed later), pick an avatar, set a 4-digit PIN.
District / Province / Country fields are optional but useful for multiple teachers.
Log in as the teacher → tap 🏦 in the top bar → re-enter PIN → fill in bank name, account name, account number → Save.
Tap 📱 to add phone, LINE ID, and email for payment notifications.
Students see bank details when they tap "Top up". Email is used for payment notifications via GAS.
Log in as the teacher → tap ＋ → enter a unique username (nickname only, no full names), choose symbol and color, set package size (10 / 15 / 20 classes), optionally set the price per package → Save.
Students log in by entering their username on the student screen. They set their own PIN on first login.
GAS enables two features: email notifications when students submit payment, and Google Sheets export of all class data.
1. Open a Google Sheet → Extensions → Apps Script
2. Tap "How to set up →" in the Google Sheets export box above → copy the code → paste it into Apps Script → Save
3. Deploy → New deployment → Web app → set access to Anyone → Deploy → copy the URL
4. Paste the URL in the Google Sheets export box above → Export ↗ to test
Without GAS, the app still works — students just can't send payment notifications automatically.
🔧 Maintenance guide
1. Ask Claude to make changes → download new index.html
2. Go to github.com/yogawithjib/tally
3. Click on index.html → Edit (pencil icon) → drag and drop your new file → Commit changes
4. Wait ~1 minute → live at yogawithjib.github.io/tally
All student and teacher data lives in Firebase — updating the app never deletes data.
Rules expire January 2027 — renew by December 2026.
1. Firebase console → Firestore Database → Rules tab
2. Find the date 2027-01-01 → change to 2028-01-01
3. Click Publish
If rules expire, the app goes read-only — students can still view but teachers can't save changes.
Teachers cannot reset their own PIN remotely.
1. Scroll down to the Teachers section below → tap 🗑 next to the teacher
2. Confirm deletion (this removes the account but not the student data)
3. Teacher creates a new account using the same username
4. All their students are automatically re-linked
The teacher will need to re-enter their bank details after re-creating their account.
The teacher handles this — not you.
Teacher logs in → taps the 📋 history button next to the student → taps 🔒 Reset PIN
The student will be prompted to set a new PIN on their next login.
Students can also change their own PIN by tapping 🔒 Change PIN on their student card.
If notifications stop working or you want to update the script:
1. Go to script.google.com → open the Yoga with Jib project
2. Tap "How to set up →" above → copy the latest code
3. Replace all code in Apps Script → Save → Deploy → Manage deployments → Edit → New version → Deploy
4. Copy the new URL → paste it in the Google Sheets export box above → save
The teacher also needs their email set in 📱 Contact info for notifications to reach them.
⚠️ This permanently deletes the teacher account and all their students' data.
1. Export to Google Sheets first if you need a record
2. Scroll down to Teachers → tap 🗑 next to the teacher name → confirm
This cannot be undone. Student PINs and history are gone permanently.
The Firebase API key is currently restricted to yogawithjib.github.io/* — this is correct and secure.
If the app stops syncing to Firebase:
1. Go to Google Cloud console → API Key → check restrictions
2. Make sure yogawithjib.github.io/* is in the allowed referrers
3. If you change the GitHub Pages URL, update this restriction to match
The app works offline using localStorage even if Firebase is unreachable.
Data is stored in Firebase Firestore (cloud) and backed by localStorage on each device.
For a manual backup:
1. Paste your GAS URL in the box above → tap Export ↗
2. This writes all teachers and students to your Google Sheet
3. You can also view raw data at Firebase console → Firestore → teachers collection
Recommended: export to Sheets once a month as a safety net.
